## Feature
- 支持翻译文本懒加载，或者急性加载；
- 支持切换语言时，不刷新页面自动更新翻译文本；
- 支持翻译文本中带有参数；
- 支持设置翻译文本加载失败时的重试次数；
- 支持翻译文本中带有组件的复杂场景；

## 启动demo项目
- 安装依赖：
```bash
npm i
```
- 编译ng-trans库
```bash
npm run build:lib
```
- 启动demo项目
```bash
npm start
```
