## Feature
- 支持翻译文本懒加载，或者急性加载；
- 支持切换语言时，不刷新页面自动更新翻译文本；
- 支持设置翻译文本加载失败时的重试次数；
- 支持翻译文本中带有参数；
- 支持翻译文本中带有组件的复杂场景；

## Installation
```bash
$ npm i @bigbear713/ng-trans
// or
$ yarn add @bigbear713/ng-trans
```

## 启动demo项目
- 安装依赖：
```bash
npm i
```

- 编译ng-trans库
```bash
npm run build:lib
```

- 启动demo项目
```bash
npm start
```

- 部署demo
```bash
npm run build
```

- 启动带有SSR的demo项目
```bash
npm run dev:ssr
```

- 部署带有SSR的demo项目
```bash
npm run build:ssr

npm run serve:ssr
```
